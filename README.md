# Mobil UI Tutarlılık Test Otomasyon Sistemi

Bu proje, mobil uygulamaların farklı cihaz ve çözünürlüklerdeki UI (Kullanıcı Arayüzü) tutarlılığını test etmek ve raporlamak amacıyla tasarlanmış modern bir dashboard arayüzüdür.

 **Canlı Demo:** [https://bberki.github.io/mobile-ui-test/](https://bberki.github.io/mobile-ui-test/)

##  Proje Hakkında

Bu panel, Appium veya benzeri test otomasyon araçlarının çıktılarını görselleştirmek, test senaryolarını yönetmek ve cihaz profillerini incelemek için geliştirilmiş bir React prototipidir. Kullanıcı dostu arayüzü sayesinde test süreçlerinin takibini kolaylaştırır.

## Özellikler

* **Dashboard:** Toplam senaryo, başarılı testler ve aktif cihaz sayılarını içeren özet ekranı.
* **Senaryo Yönetimi:** Yeni test senaryoları oluşturma, düzenleme ve listeleme.
* **Test Simülasyonu:** Testleri arayüz üzerinden başlatma ve gerçek zamanlı durum takibi (Simülasyon).
* **Cihaz Profilleri:** Farklı çözünürlük (HD, FHD, 2K) ve DPI değerlerine sahip cihaz tanımları.
* **Detaylı Raporlama:** Başarılı, hatalı ve uyarı veren testlerin görsel analizi.

## Kullanılan Teknolojiler

Bu proje, kurulum gerektirmeyen modern web teknolojileri kullanılarak **Single File Component (Tek Dosya Bileşeni)** yapısında geliştirilmiştir:

* **React:** UI bileşenleri ve durum yönetimi için.
* **Tailwind CSS:** Hızlı ve modern stillendirme için.
* **Lucide React:** Modern ikon seti için.
* **Babel (Standalone):** Tarayıcı içinde JSX derlemesi için.

## Kurulum ve Çalıştırma

Proje tek bir HTML dosyasından oluştuğu için herhangi bir paket kurulumuna (npm install vb.) ihtiyaç duymaz.

**Local'de çalıştırmak için:**
1.  Bu repoyu klonlayın veya indirin.
2.  `index.html` dosyasını favori tarayıcınızda (Chrome, Edge, Firefox) açın.
3.  İnternet bağlantınızın olduğundan emin olun (CDN kütüphaneleri için).
